Änderungen
Folgende Änderungen sind durchzuführen:

1.) Heruntergeladene Datei (ewige_tipptabelle.rar) entpacken

2.) Dateien auf Server kopieren
ewigetippspieltabelle.php -> ins LMO-Root
ewiggraph.php -> ins LMO-Root
ewigpopop -> ins LMO-Root

img/*.png -> lmo/img/*.png

3.) Verzeichnis erstellen:
"archiv" in lmo/addon/tipp/ erstellen  ->  lmo/addon/tipp/archiv

Wichtiger Hinweis: In diesem neuen Ordner lmo/addon/tipp/archiv werden die archivierten Auswert-Dateien von nun an gespeichert.
Nach Ende einer Tipp-Saison sollte die Gesamtauswertungs-Datei (lmo/addon/tipp/tipps/auswert/gesamt.aus) in den neu erstellten Ordner kopiert und nach folgendem Schema umbenannt werden: gesamt.aus -> gesamtXXYY.aus - also z.B. gesamt0809.aus
Die Datei muss so benannt werden, da das Skript anhand der Saison XXYY die Sortierung vornimmt und auch Ausgaben darauf aufbauen.
Sollte eine Datei nicht entsprechend umbenannt wurden sein, so funktioniert die Auswertung dennoch, aber die entsprechende Datei wird nicht berücksichtigt.

Einstellmöglichkeiten
Im Skript (ewigetippspieltabelle.php) können ab Zeile 31 bis 104 diverse Einstellungen vorgenommen werden.

Es kann u.a. eingestellt werden:
  • Tippspieler, die noch keinen einzigen Tipp abgegeben haben, ausgeblendet werden können
  • Begrenzung der Anzahl der anzuzeigenden Tipper
  • Auswertung nur für eingeloggte Tippspieler sichtbar
  • Länge der Tippspieler-Usernamen einstellbar
  • Seitentitel
  • Spaltenüberschriften
  • Farben (u.a. Platz 1-3) und Schriftgrößen
  • ob Sortierung möglich ist
Wem das nicht genug ist, der kann ab Zeile 471 direkt am CSS-Code Änderungen vornehmen.

Das Ein- und Ausblenden, sowie die mouseover-Effekte werden durch das Ajax-Framework JQuery realisiert.
Standardmäßig wird im Skript die JQuery-Datei vom Google-Server eingebunden.
Wer seine lokale JQuery-Datei einbinden möchte muss die Zeile
<script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/jquery/1.4.1/jquery.min.js"></script>
durch z.B. folgende ersetzen:
<script type="text/javascript" src="../js/jquery.min.js"></script>
